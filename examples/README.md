# Examples for Gift Recommendation Bot

This directory contains example scripts demonstrating how to use this project.

## Quick Demo

```bash
python examples/demo.py
```

## What the Demo Shows

- **`generate_recommendations()`** — Generate gift recommendations based on parameters.
- **`get_gift_details()`** — Get detailed information about a specific gift idea.
- **`compare_prices()`** — Get price comparison information for a gift.
- **`load_wishlists()`** — Load all wishlists.
- **`save_wishlists()`** — Save wishlists.

## Prerequisites

- Python 3.10+
- Ollama running with Gemma 4 model
- Project dependencies installed (`pip install -e .`)

## Running

From the project root directory:

```bash
# Install the project in development mode
pip install -e .

# Run the demo
python examples/demo.py
```
